**https://leetcode.cn/problems/count-equal-and-divisible-pairs-in-an-array/description/**

- Solution by enumeration
```
public class Solution {
    public int CountPairs(int[] nums, int k) {
        var n = nums.Length;
        var ans = 0;
        for (var i = 0; i < n; i++)
        {
            for (var j = i + 1; j < n; j++)
            {
                if (nums[i] == nums[j] && (i * j) % k == 0)
                    ans++;
            }
        }
        return ans;
    }
}
```
**Python**
```
class Solution:
    def countPairs(self, nums: List[int], k: int) -> int:
        n, ans = len(nums), 0
        for i in range(n):
            for j in range(i + 1, n):
                if nums[i] == nums[j] and (i * j) % k == 0:
                    ans += 1
        return ans
```

- Recommendation by enumerating nums pair
```
class Solution:
    def countPairs(self, nums: List[int], k: int) -> int:
        n = len(nums)
        res = 0
        for i in range(n - 1):
            for j in range(i + 1, n):
                if (i * j) % k == 0 and nums[i] == nums[j]:
                    res += 1
        return res
```
