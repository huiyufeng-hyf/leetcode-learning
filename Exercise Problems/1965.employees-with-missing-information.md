**https://leetcode.cn/problems/employees-with-missing-information/description/**

- Solution by using CTE in MS Sql
```
/* Write your T-SQL query statement below */
WITH CTE AS (
    SELECT employee_id
    FROM Employees
    WHERE employee_id NOT IN (
        SELECT employee_id FROM Salaries
    )

    UNION

    SELECT employee_id
    FROM Salaries
    WHERE employee_id NOT IN (
        SELECT employee_id FROM Employees
    )
)
SELECT employee_id
FROM CTE 
ORDER BY employee_id

```

- Recommendation in Pandas
```
import pandas as pd

def find_employees(employees: pd.DataFrame, salaries: pd.DataFrame) -> pd.DataFrame:
    df = pd.merge(employees, salaries, how='outer', on='employee_id')
    df = df[df.isna().any(axis=1)][['employee_id']]
    return df
    
```

- Recommendation in Sql
```
/* Write your T-SQL query statement below */
select employee_id
from (
    select employee_id from employees
    union all
    select employee_id from salaries
) as t
group by employee_id
having count(employee_id) = 1
order by employee_id
```
