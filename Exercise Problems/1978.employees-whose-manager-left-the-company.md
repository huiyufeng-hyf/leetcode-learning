**https://leetcode.cn/problems/employees-whose-manager-left-the-company/description/**

- Solution in MS Sql
```
/* Write your T-SQL query statement below */
select employee_id
from Employees
where salary < 30000 and manager_id not in (
    select employee_id from Employees
)
order by employee_id
```

- Recommendation in Pandas by left join and null value
```
import pandas as pd

def find_employees(employees: pd.DataFrame) -> pd.DataFrame:
    lt_30000 = employees[employees["salary"] < 30000]
    df = pd.merge(lt_30000, employees[["employee_id", "salary"]], how="left", left_on="manager_id", right_on="employee_id")
    df = df.dropna(subset="manager_id")
    return df.loc[df["employee_id_y"].isna(), ["employee_id_x"]].rename(columns={"employee_id_x": "employee_id"}).sort_values("employee_id")
    
```
