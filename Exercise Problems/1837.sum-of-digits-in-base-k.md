https://leetcode.cn/problems/sum-of-digits-in-base-k/description/ 

- Solution by Math
```
public class Solution {
    public int SumBase(int n, int k) {
        var res = 0;
        while (n > 0)
        {
            res += n % k;
            n /= k;
        }
        return res;
    }
}
```
**Python**
```
class Solution:
    def sumBase(self, n: int, k: int) -> int:
        res = 0
        while n > 0:
            res += n % k
            n = n // k
        return res
```
