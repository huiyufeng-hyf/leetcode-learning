**https://leetcode.cn/problems/build-array-from-permutation/description/**

- Solution by using new array
```
public class Solution {
    public int[] BuildArray(int[] nums) {
        var n = nums.Length;
        var ans = new int[n];
        for (var i = 0; i < n; i++)
        {
            ans[i] = nums[nums[i]];
        }
        return ans;
    }
}
```
**Python**
```
class Solution:
    def buildArray(self, nums: List[int]) -> List[int]:
        n = len(nums)
        ans = [0] * n
        for i in range(n):
            ans[i] = nums[nums[i]]
        return ans
        
```

- Recommendation 1 by building array
```
class Solution:
    def buildArray(self, nums: List[int]) -> List[int]:
        n = len(nums)
        return [nums[nums[_]] for _ in range(n)]
        
```

class Solution:
    def buildArray(self, nums: List[int]) -> List[int]:
        n = len(nums)
        for i in range(n):
            nums[i] += 1000 * (nums[nums[i]] % 1000)
        
        for i in range(n):
            nums[i] //= 1000
        return nums
