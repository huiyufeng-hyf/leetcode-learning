**https://leetcode.cn/problems/check-if-string-is-a-prefix-of-array/description/**

- Solution by simulation
```
public class Solution {
    public bool IsPrefixString(string s, string[] words) {
        if (s.Length < words[0].Length)
            return false;

        var sb = new StringBuilder();
        foreach (var word in words)
        {
            sb.Append(word);
            if (s.Equals(sb.ToString()))
                return true;
        }
        return false;
    }
}
```
**Python**
```
class Solution:
    def isPrefixString(self, s: str, words: List[str]) -> bool:
        if len(s) < len(words[0]):
            return False
        
        temp = ""
        for i in range(len(words)):
            temp += "".join(words[i])
            if s == temp:
                return True
        return False
        
```

- Recommendation by comparing each character
```
class Solution:
    def isPrefixString(self, s: str, words: List[str]) -> bool:
        i, n = 0, len(s)
        for word in words:
            for ch in word:
                if i < n and ch == s[i]:
                    i += 1
                else:
                    return False
            if i == n:
                return True
        return False
        
```
