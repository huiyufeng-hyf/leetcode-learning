**https://leetcode.cn/problems/find-subsequence-of-length-k-with-the-largest-sum**

- Solution by copilot
```
public class Solution {
    public int[] MaxSubsequence(int[] nums, int k) {
        return Enumerable.Range(0, nums.Length)
            .OrderByDescending(i => nums[i])
            .Take(k)
            .OrderBy(i => i)
            .Select(i => nums[i])
            .ToArray();
    }
}

**Python**
```
class Solution:
    def maxSubsequence(self, nums: List[int], k: int) -> List[int]:
        indices = sorted(range(len(nums)), key=lambda i: nums[i], reverse=True)[:k]
        return [nums[i] for i in sorted(indices)]
```

class Solution:
    def maxSubsequence(self, nums: List[int], k: int) -> List[int]:
        n = len(nums)
        vals = [[i, nums[i]] for i in range(n)]
        vals.sort(key=lambda x: -x[1])
        vals = sorted(vals[:k])
        res = [val for idx, val in vals]
        return res
```
