Leetcode link: https://leetcode.cn/problems/biggest-single-number/description/ 

- Solution 1:
```
/* Write your T-SQL query statement below */



WITH cte AS

(SELECT num

FROM MyNumbers

GROUP BY num

HAVING COUNT(*) = 1)

SELECT MAX(num) AS num

FROM cte

ORDER BY num DESC
```
