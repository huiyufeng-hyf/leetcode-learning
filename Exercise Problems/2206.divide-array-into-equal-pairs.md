**https://leetcode.cn/problems/divide-array-into-equal-pairs/description/**

- Solution by dictionary
```
public class Solution {
    public bool DivideArray(int[] nums) {
        var n = nums.Length;
        var dict = new Dictionary<int, int>();
        foreach (var num in nums)
        {
            if (dict.ContainsKey(num))
                dict[num]++;
            else
                dict[num] = 1;
        }
        if (dict.Any(i => i.Value % 2 == 1))
            return false;
        return true;
    }
}
```
**Python**
```
class Solution:
    def divideArray(self, nums: List[int]) -> bool:
        num_cnt = Counter(nums)
        for x in num_cnt.values():
            if x % 2 == 1:
                return False
        return True
```

- Recommendation by hashset
```
class Solution:
    def divideArray(self, nums: List[int]) -> bool:
        freq = Counter(nums)
        return all(f % 2 == 0 for f in freq.values())
```
