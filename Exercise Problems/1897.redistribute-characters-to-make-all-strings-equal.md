**https://leetcode.cn/problems/redistribute-characters-to-make-all-strings-equal/description/**

- Solution by copilot
```
public class Solution {
    public bool MakeEqual(string[] words) {
        var n = words.Length;
        var counts = new int[26];
        foreach (var word in words)
        {
            foreach (var ch in word)
            {
                var idx = ch - 'a';
                counts[idx]++;
            }
        }
        
        for (var i = 0; i < 26; i++)
        {
            if (counts[i] % n != 0)
                return false;
        }
        return true;
    }
}
```
**Python**
```
class Solution:
    def makeEqual(self, words: List[str]) -> bool:
        n = len(words)
        counts = [0] * 26

        for word in words:
            for ch in word:
                idx = ord(ch) - ord('a')
                counts[idx] += 1
        
        for i in range(26):
            if counts[i] % n != 0:
                return False
        return True
        
```

- Recommendation by statistic frequency
```
class Solution:
    def makeEqual(self, words: List[str]) -> bool:
        cnt = [0] * 26
        n = len(words)
        for wd in words:
            for ch in wd:
                cnt[ord(ch)-ord('a')] += 1
        return all(k % n == 0 for k in cnt)
```
