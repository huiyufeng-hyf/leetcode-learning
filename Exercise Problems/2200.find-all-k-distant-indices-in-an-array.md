**https://leetcode.cn/problems/find-all-k-distant-indices-in-an-array/description/**

- Solution by enumeration
```
public class Solution {
    public IList<int> FindKDistantIndices(int[] nums, int key, int k) {
        var n = nums.Length;
        var res = new HashSet<int>();
        for (var i = 0; i < n; i++)
        {
            for (var j = i; j < n; j++)
            {
                if (Math.Abs(i - j) <= k && (nums[j] == key || nums[i] == key))
                {
                    res.Add(i);
                    res.Add(j);
                }
            }
        }
        return res.OrderBy(i => i).ToList();
    }
}

**Python**
```
class Solution:
    def findKDistantIndices(self, nums: List[int], key: int, k: int) -> List[int]:
        n = len(nums)
        res = set()
        for i in range(n):
            for j in range(n):
                if abs(i - j) <= k and (nums[i] == key or nums[j] == key):
                    res.add(i)
                    res.add(j)
        return sorted(res)

- Recommendation 2 by one time enumeration
```
class Solution:
    def findKDistantIndices(self, nums: List[int], key: int, k: int) -> List[int]:
        res = []
        r = 0
        n = len(nums)
        for j in range(n):
            if nums[j] == key:
                l = max(r, j - k)
                r = min(n - 1, j + k) + 1
                for i in range(l, r):
                    res.append(i)
        return res
```
