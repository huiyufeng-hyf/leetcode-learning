**https://leetcode.cn/problems/find-target-indices-after-sorting-array/description/**

- Solution by enumeration
```
public class Solution {
    public IList<int> TargetIndices(int[] nums, int target) {
        var res = new List<int>();
        Array.Sort(nums);
        for (var i = 0; i < nums.Length; i++)
        {
            if (nums[i] == target)
                res.Add(i);
        }
        return res;
    }
}
```
**Python**
```
class Solution:
    def targetIndices(self, nums: List[int], target: int) -> List[int]:
        res = []
        nums.sort()
        for i, n in enumerate(nums):
            if n == target:
                res.append(i)
        return res
```

- Recommendation by counting directly
```
class Solution:
    def targetIndices(self, nums: List[int], target: int) -> List[int]:
        cnt1 = 0
        cnt2 = 0
        for num in nums:
            if num < target:
                cnt1 += 1
            elif num == target:
                cnt2 += 1
        res = list(range(cnt1, cnt1 + cnt2))
        return res
```
