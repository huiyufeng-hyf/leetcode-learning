**Python**

```
class Solution:
    def checkString(self, s: str) -> bool:
        return "ba" not in s
```

- Recommendation 1 by comparing index
```
class Solution:
    def checkString(self, s: str) -> bool:
        n = len(s)
        lasta = -1
        firstb = n
        for i in range(n):
            if s[i] == 'a':
                lasta = max(lasta, i)
            else:
                firstb = min(firstb, i)
        return lasta < firstb
```

```
class Solution:
    def checkString(self, s: str) -> bool:
        return s.find("ba") == -1
```
