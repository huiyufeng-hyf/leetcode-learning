Leetcode link: https://leetcode.cn/problems/power-of-four/description/ 

# Solution 1:
```
public class Solution {
    public bool IsPowerOfFour(int n) {
        if (n == 0) return false;
        while (n % 4 == 0)
        {
            n /= 4;
        }

        return n == 1;
    }
}
```

# Solution 2: (Recommendation)
```
public class Solution {
    public bool IsPowerOfFour(int n) {
        return n > 0 && (n & (n - 1)) == 0 && n % 3 == 1;
    }
}
```
