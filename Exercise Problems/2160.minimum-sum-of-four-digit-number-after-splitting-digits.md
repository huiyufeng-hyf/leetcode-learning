**https://leetcode.cn/problems/minimum-sum-of-four-digit-number-after-splitting-digits/description/**

- Solution by math
```
public class Solution {
    public int MinimumSum(int num) {
        var digits = new int[4];
        var idx = 0;
        while (num > 0)
        {
            var d = num % 10;
            digits[idx] = d;
            idx++;
            num /= 10;
        }
        Array.Sort(digits);
        return (digits[0] * 10 + digits[3]) + (digits[1] * 10 + digits[2]);
    }
}
```
**Python**
```
class Solution:
    def minimumSum(self, num: int) -> int:
        digits = [0] * 4
        idx = 0
        while num > 0:
            d = num % 10
            digits[idx] = d
            idx += 1
            num //= 10
        digits.sort()
        return (digits[0] * 10 + digits[3]) + (digits[1] * 10 + digits[2])
```

- Recommendation by greedy
```
class Solution:
    def minimumSum(self, num: int) -> int:
        digits = []
        while num:
            digits.append(num % 10)
            num //= 10
        digits.sort()
        return 10 * (digits[0] + digits[1]) + digits[2] + digits[3]
```
