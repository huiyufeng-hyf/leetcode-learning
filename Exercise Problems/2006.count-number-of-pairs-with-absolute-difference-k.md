**https://leetcode.cn/problems/count-number-of-pairs-with-absolute-difference-k/description/**

- Solution by enumeration
```
public class Solution {
    public int CountKDifference(int[] nums, int k) {
        var n = nums.Length;
        var cnt = 0;
        for (var i = 0; i < n - 1; i++)
        {
            for (var j = i + 1; j < n; j++)
            {
                if (Math.Abs(nums[i] - nums[j]) == k)
                    cnt++;
            }
        }
        return cnt;
    }
}
```
**Python**
```
class Solution:
    def countKDifference(self, nums: List[int], k: int) -> int:
        n, cnt = len(nums), 0
        for i in range(n - 1):
            for j in range(i + 1, n):
                if abs(nums[i] - nums[j]) == k:
                    cnt += 1
        return cnt
        
```

- Recommendation by hashset + one time enumeration
```
class Solution:
    def countKDifference(self, nums: List[int], k: int) -> int:
        res = 0
        cnt = Counter()
        for num in nums:
            res += cnt[num - k] + cnt[num + k]
            cnt[num] += 1
        return res
        
```
