- Solution by MS SQL
```
/* Write your T-SQL query statement below */
SELECT employee_id, 
    CASE 
        WHEN employee_id % 2 = 1 AND name NOT LIKE 'M%' THEN salary
        ELSE 0
    END AS 'bonus'
FROM Employees
ORDER BY employee_id

```

- Solution by copilot
```
import pandas as pd

def calculate_special_bonus(employees: pd.DataFrame) -> pd.DataFrame:
    # Assuming df is your DataFrame with columns: employee_id, name, salary
    employees['bonus'] = employees.apply(
        lambda row: row['salary'] if (row['employee_id'] % 2 == 1 and not row['name'].startswith('M')) else 0,
        axis=1
    )

    # Select and sort
    result = employees[['employee_id', 'bonus']].sort_values('employee_id')
    return result
```
